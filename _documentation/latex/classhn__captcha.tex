You should have received a copy of the GNU General Public License along with this program; if not, write to the Free Software Foundation, Inc., 59 Temple Place, Suite 330, Boston, MA 02111-1307 USA Tabsize: 4 This class generates a picture to use in forms that perform CAPTCHA test (Completely Automated Public Turing to tell Computers from Humans Apart). After the test form is submitted a key entered by the user in a text field is compared by the class to determine whether it matches the text in the picture.

The class is a fork of the original released at www.phpclasses.org by Julien Pachet with the name ocr\_\-captcha.

The following enhancements were added:

\begin{itemize}
\item Support to make it work with GD library before version 2\item Hacking prevention\item Optional use of Web safe colors\item Limit the number of users attempts\item Display an optional refresh link to generate a new picture with a different key without counting to the user attempts limit verification\item Support the use of multiple random TrueType fonts\item Control the output image by only three parameters: number of text characters and minimum and maximum size preserving the size proportion\item Preserve all request parameters passed to the page via the GET method, so the CAPTCHA test can be added to existing scripts with minimal changes\item Added a debug option for testing the current configuration\end{itemize}


All the configuration settings are passed to the class in an array when the object instance is initialized.

The class only needs two function calls to be used: \doxyref{display\_\-form()}{S.}{classhn__captcha_4fe7655f9890c7271b4d85a0ef7849b6} and \doxyref{validate\_\-submit()}{S.}{classhn__captcha_58cf521777ae952c7e3c1f21850f44cf}.
